# 🌐 常见 NGS 测序平台原理与差异详解

---

## 🧬 一、Illumina 平台（主流二代测序）

### **基本原理：边合成边测序（Sequencing by Synthesis, SBS）**
Illumina 是目前应用最广泛的二代测序平台，其核心思想是：
> 通过在固相表面进行 **可逆终止子化学反应** 实现“边合成、边检测”的测序。

### **工作流程**
1. **文库制备**：DNA 片段化并加上接头（Adaptor）。
2. **桥式扩增（Bridge Amplification）**：在 flow cell 上形成 **DNA cluster（克隆簇）**。
3. **测序反应**：
   - 使用 **带有荧光标记和可逆终止基团的 dNTPs**；
   - 每次掺入一个碱基后发出荧光信号；
   - 摄像记录颜色 → 确定碱基；
   - 去除荧光基团 → 继续下一循环。
4. **数据输出**：得到高质量短读长序列（50–300 bp）。

### **优缺点**
| 优点 | 缺点 |
|------|------|
| 准确率高（>99.9%） | 读长短（50–300 bp） |
| 成本低、通量高 | 无法直接检测大片段结构变异 |
| 软件生态成熟 | 需 PCR 扩增，存在 GC 偏倚 |

## ⚡ 二、Ion Torrent 平台（半导体测序）

### **基本原理：检测 pH 变化**
Ion Torrent 通过检测 **DNA 聚合反应中释放的 H⁺ 离子** 来识别碱基，而非荧光信号。

### **工作流程**
1. DNA 文库制备并固定在微珠上；
2. 每个反应孔中放入单一模板；
3. 依次加入四种 dNTP；
4. 若掺入正确碱基，会释放 H⁺；
5. 半导体传感器检测 pH 变化 → 判定碱基。

### **优缺点**
| 优点 | 缺点 |
|------|------|
| 无需荧光检测、速度快 | 同一碱基连续重复（如 AAAA）易误判 |
| 成本低、设备简单 | 准确率低于 Illumina（约 98%） |
| 适合小 panel、临床诊断 | 读长中等（200–400 bp） |

## 🧪 三、华大 DNBSEQ 平台（DNA 纳米球测序）

### **基本原理：DNA Nanoball Sequencing（DNBSEQ）**
BGI/MGI 自主研发的二代测序技术，基于改良的 **SBS 原理 + 滚环扩增（RCA）**，  
形成大量无 PCR 偏倚的 **DNA 纳米球（DNB）**，在阵列上进行荧光检测。

### **工作流程**
1. **文库制备**：DNA 片段化并加接头，形成环状模板；
2. **滚环扩增 (RCA)**：生成上万个拷贝的 DNA 纳米球；
3. **固定上机**：DNB 加载到 Patterned Array；
4. **cPAS 化学测序**（Combinatorial Probe-Anchor Synthesis）：
   - 探针锚定到模板；
   - 荧光标记探针识别碱基；
   - 光学检测、洗脱、循环。

### **技术优势**
| 特性 | 说明 |
|------|------|
| **无 PCR 偏倚** | RCA 是物理扩增，避免 GC 偏倚 |
| **高准确率** | 信号强、背景噪音低 |
| **低成本高通量** | 每 Gb 成本低于 Illumina |
| **兼容性强** | 支持 WGS、RNA-seq、mNGS、tNGS 等 |
| **国产化优势** | 成本可控，适合大规模项目 |

### **常见机型**
| 型号 | 特点 |
|------|------|
| **MGISEQ-2000 / DNBSEQ-G400** | 中高通量 |
| **DNBSEQ-T7** | 超高通量，可与 NovaSeq 对标 |
| **DNBSEQ-G99** | 快速测序（<24小时出结果） |
| **DNBSEQ-T20×2** | 年产数据达 50Tb 以上 |

### **应用场景**
- 临床 **tNGS 检测**（感染、肿瘤 panel）  
- **宏基因组 mNGS 分析**  
- **转录组 / 单细胞测序**  
- **群体遗传与重测序研究**

## 🧫 四、PacBio 平台（SMRT Sequencing，三代测序）

### **基本原理：单分子实时测序（Single Molecule Real-Time, SMRT）**
PacBio 直接观测单个 DNA 分子的合成过程，无需 PCR 扩增。

### **工作流程**
1. DNA 加上发夹接头形成环状分子（SMRTbell）；
2. 放入纳米孔腔（Zero-Mode Waveguide, ZMW）；
3. 聚合酶在底部固定；
4. 每掺入一个荧光标记碱基时发光；
5. 实时检测光信号 → 得到碱基序列。

### **最新技术**
- **HiFi Reads（高保真读长）**：多次循环读取同一分子后共识校正，兼具高准确率与长读长。

### **性能指标**
| 参数 | 典型值 |
|------|--------|
| 读长 | 10–25 kb（HiFi 可达 15–20 kb） |
| 准确率 | >99.8%（HiFi 模式） |
| 应用 | 全基因组组装、结构变异、Iso-Seq 转录组分析 |

## 🌊 五、Oxford Nanopore 平台（纳米孔测序）

### **基本原理：电流信号测序**
DNA 分子穿过蛋白质纳米孔（Nanopore）时，不同碱基导致 **离子电流强度变化**。系统实时检测该变化信号并解码为碱基。

### **工作流程**
1. DNA 通过纳米孔；
2. 电流部分阻断；
3. 不同碱基组合改变电流特征；
4. 实时记录电流变化；
5. 算法（Basecalling）将信号转化为序列。

### **设备类型**
- **MinION**：便携式设备  
- **PromethION**：高通量测序平台  
- **Flongle**：快速低通量测序  

### **优缺点**
| 优点 | 缺点 |
|------|------|
| 超长读长（可达 2 Mb） | 错误率相对较高（改进至 <1–3%） |
| 实时测序、无需 PCR | Basecalling 计算量大 |
| 可直接检测甲基化修饰 | 数据分析复杂度高 |

## 🚀 七、最新测序技术趋势（2023–2025）

| 技术方向 | 新进展 |
|-----------|--------|
| **Illumina NovaSeq X 系列** | 使用 XLEAP-SBS 化学，成本降至 <$2/Gb |
| **PacBio Revio 系统** | 提升通量，HiFi 成主流长读长方案 |
| **Nanopore Duplex Sequencing** | 双链测序，准确率提升至 99.9% |
| **BGI DNBSEQ T20×2** | 超级高通量、低成本 WGS |
| **合成长读长（Synthetic Long Read）** | 结合短读长测序与 barcode 技术 |
| **单细胞 + 空间转录组整合测序** | 多维度解析细胞空间异质性 |

## 🧭 八、测序技术演进趋势

| 阶段 | 技术 | 特征 | 代表平台 |
|------|------|------|----------|
| 二代测序（SBS） | 化学合成测序 | 高准确率、短读长 | Illumina、DNBSEQ |
| 改良二代 | 电信号检测 | 快速、低成本 | Ion Torrent |
| 三代测序 | 单分子实时检测 | 长读长、高错误率 | PacBio、Nanopore |
| 三代改进 | 共识或双链校正 | 长读长 + 高准确率 | PacBio HiFi、Nanopore Duplex |
| 新兴方向 | 空间、多组学整合 | 一次测序获取多维信息 | PacBio、ONT、BGI |

## 🧠 九、一句话总结

| 平台 | 一句话总结 |
|-------|-------------|
| **Illumina** | 全球最成熟的短读长测序标准 |
| **Ion Torrent** | 快速、简便的电信号测序平台 |
| **BGI DNBSEQ** | 国产高通量、低成本、无 PCR 偏倚新一代平台 |
| **PacBio HiFi** | 高保真长读长的结构变异利器 |
| **Nanopore** | 实时、超长读长、可检测甲基化 |


---

> 📘 **总结**：  
> Illumina 与华大 DNBSEQ 属于短读长高准确率平台；  
> PacBio 与 Nanopore 属于长读长三代平台；  
> Ion Torrent 适用于快速小 panel 检测。  
> 当前趋势是向 **长读长、高准确率、多组学整合** 方向发展。
